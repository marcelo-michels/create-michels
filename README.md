# node-ts

A basic Node JS template created with `npm create michels` for startup projects with:

- Typescript
- Husky
- Jest
- Prettier
- Lint
- VS Code Settings
- Github Actions

## Usage

Install Dependencies

```sh
npm install
```

Devevelopment with live reload

```sh
npm run dev
```

Test

```sh
npm run test
```

Test Coverage

```sh
npm run test:coverage
```

Update upstream

```
node scripts/update-upstream.js
```
