# node-ts

A basic Node JS template for startup projects with:

- Typescript
- Nodemon
- Husky
- Jest
- Prettier
- Lint
- VS Code Settings
- Github Actions

## Usage

Install Dependencies

```sh
npm install
```

Devevelopment with live reload

```sh
npm run dev
```

Test

```sh
npm run test
```

Test Coverage

```sh
npm run test:coverage
```
